About the Developer

Hi, I'm Jonas Bratschi. I switched to Mac some time ago and always searched for a simple PDF viewer. But all I found were either extremely advanced PDF tools that cost a fortune, or free bloatware that just want to sell you a subscription.

So I thought of this a long time ago and at some point I just prompt engineered it into existence (to quote Peter Steinberger about OpenClaw). If you enjoy using SimplePDF, consider supporting my work!
